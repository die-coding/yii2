## Cara Memasang

* Melalui console

```
composer require --prefer-dist diecoding/yii2-core "*"
```

* Melalui `composer.json`

1. Tambahkan pada baris `require`

```
"diecoding/yii2-core": "*"
```

2. Kemudian jalankan

```
composer update
```
